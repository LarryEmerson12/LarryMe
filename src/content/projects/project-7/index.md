## MacOS React

This open source project was created to replicate [MacOS](https://www.apple.com/mac) but with standard web technologies like HTML 🟠, CSS #️⃣, and JS 🟡.

🔗🔗 [macos-react](https://macos-react-two.vercel.app/)

### 🤔 Why

I've always wanted to create a operating system. I love MacOS 🍎 for the design. So here it is.

### 🥪 Stack

- 🧩 Framework - NextJS ⚛️
- 📦 Bundler - Turbopack 🧊
- 🍱 Component Library - None 🎉
- 🎨 Styling Solution - Tailwind CSS 🦄
